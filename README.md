# wow_hong_golang

这是一个为魔兽世界玩家制作的游戏工具，该工具提供给玩家一个分享游戏中宏命令的平台，也同时提供给玩家以搜索功能。

# 功能
1. 快速创建

  提供给对游戏中宏命令感兴趣，却不懂的玩家，融合搜索和自动生成的功能，返回给玩家一个命令列表以供选择。

2. 手动组合

  提供给熟悉游戏的玩家，通过工具，手动组合出自己所需的命令。

3. 分享及搜索

  提供给玩家一个分享自己收集的宏命令的功能，同时也可以搜索其他玩家分享的宏命令。
